The following are some helpful links which may assist those conducting similar research, or who just want to learn more about Android related malware and repackaged Android applications.

  • Darwin Project. Contains information about over 30,000 Android applications such as overprivledges, vulneranability risk assmesment, discoverd defects, adherance to coding standards and discovered code clones.
  • Contagio Mobile Minidump. Contains numerous examples of malcious Android applications, including many examples of repackaged applications.
  • Malware Genome Project. Contains over 1,200 malware samples which cover the majority of existing Android malware families.
  • F-Droid. Open source repository for numerous Android applicatoins. Researchers may find this site interesting due to their ability to examine large amounts of Android source code from a variety of developers.
